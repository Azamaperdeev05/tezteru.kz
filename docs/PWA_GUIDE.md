# 📱 PWA (Progressive Web App) Guide

**tezteru.kz** — бұл кәдімгі веб-сайт қана емес, сондай-ақ заманауи **PWA** қолданбасы. Оны кез келген құрылғыға (iPhone, Android, Windows, macOS) орнатуға болады.

## 1. PWA негізгі мүмкіндіктері

* **📱 Орнату**: Сайтты браузер емес, жеке қолданба ретінде ашу.
* **🌐 Оффлайн жұмыс**: Мәтіндер мен негізгі интерфейс интернетсіз де жүктеледі.
* **⚡️ Жылдамдық**: Кэштеу жүйесі (`Workbox`) арқылы сайт 2-3 есе тез ашылады.
* **🎨 Теңшеу**: Мобильді құрылғылардағы статус-бар мен навигация жолағы сайттың түсіне сәйкес келеді.

## 2. Қалай орнату керек?

### Android (Chrome)

Браузердің мәзірінен **"Install app"** (Қолданбаны орнату) немесе сайттың жоғарғы жағындағы "Орнату" батырмасын басыңыз.

### iOS (Safari)

Safari браузерінен "Share" (Бөлісу) батырмасын басып, **"Add to Home Screen"** таңдаңыз.

### Windows / macOS

Мекенжай жолағының оң жағындағы "Орнату" белгішесін басыңыз.

## 3. Техникалық баптаулар (VitePWA)

Жобада `vite-plugin-pwa` қолданылады. Ол автоматты түрде `service-worker.js` және `manifest.json` файлдарын жасайды.

### Manifest конфигурациясы

`vite.config.ts` файлында PWA баптауларын көре аласыз:

```typescript
VitePWA({
  manifest: {
    name: 'Tezteru.kz',
    short_name: 'Tezteru',
    description: 'Қазақ тілінде жылдам жазуды үйрену',
    theme_color: '#323437',
    icons: [
      {
        src: 'logo.png',
        sizes: '192x192',
        type: 'image/png'
      }
      // ...басқа өлшемдер
    ]
  }
})
```
