# 🏗 Жоба Архитектурасы | Project Architecture

Бұл құжат **tezteru.kz** жобасының техникалық құрылымын, өнімділікті арттыру әдістерін және басты инженерлік шешімдерді түсіндіреді.

## 1. Технологиялық стек

* **Vite**: Жобаны жылдам құрастыру (Build) және ыстық қайта жүктеу (HMR) үшін қолданылады.
* **React 19**: Жаңа мүмкіндіктер мен жылдамдықты қамтамасыз етеді.
* **TypeScript**: Типтер қауіпсіздігі мен кодтың тазалығы үшін.
* **Tailwind CSS v4**: Заманауи стильдеу мен минималистік дизайн.
* **Firebase**: Авторизация мен деректер қорын басқару.

## 2. Өнімділік (Performance Optimization)

Теру машинасының жылдамдығы мен сайттың "қатып" қалмауы үшін келесі шешімдер енгізілді:

### ⚡️ Character Windowing

Теру кезінде барлық мәтінді рендерлеудің орнына, тек курсор айналасындағы (мысалы, алдағы және өткен 30 таңба) таңбалар ғана DOM-да белсенді болады. Бұл мыңдаған сөздерден тұратын мәтіндерде де сайтты жылдам жұмыс істетеді.

### ⚡️ requestAnimationFrame

Курсордың жылжуы мен анимациялар `requestAnimationFrame` арқылы орындалады. Бұл браузердің кадрларымен үндестіріліп, "layout thrashing" (орналасуды қайта есептеу) қателерінен сақтайды.

### ⚡️ localStorage Caching

Пайдаланушы баптаулары (дыбыс, тақырып) `Ref` арқылы Кэштелген. `handleInput` функциясында `localStorage`-қа жиі жүгінудің алдын алады.

## 3. Тақырыптар мен Стильдер

Жобада **CSS Variables** жүйесі қолданылады. Әрбір тақырып (`theme`) негізгі түстерді (`--bg-color`, `--main-color`, `--accent-color`, `--sub-color`) анықтайды.

```css
:root {
  --bg-color: #323437;
  --main-color: #e2b714;
  --accent-color: #e2b714;
  --sub-color: #646669;
}
```

## 4. Hook-тар мен Көйлектер

* `useTypingTest`: Теру логикасын басқаратын орталық Hook.
* `useGoogleOneTap`: Google авторизациясын басқару.
* `usePWAInstall`: Қолданбаны орнату логикасы.
