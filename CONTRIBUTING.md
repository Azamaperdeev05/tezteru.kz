# 🤝 Үлес қосу | Contributing to tezteru.kz

Біз ашық кодты жобаларға қолдау көрсетуге қуаныштымыз! Егер сізде идеялар немесе қателерді түзету жолдары болса, мына қадамдарды орындаңыз.

## 1. Қалай көмектесуге болады?

* **🐛 Қателерді анықтау**: Егер біздің сайтта қате тапсаңыз, [GitHub Issue](https://github.com/perdeev/tezteru.kz/issues) ашыңыз.
* **💡 Жаңа идеялар**: Егер жаңа функция немесе дизайн идеясы болса, бізге хабарласыңыз.
* **💻 Кодпен бөлісу**: Жаңа мүмкіндіктерді немесе қателерді өзіңіз жазсаңыз, **Pull Request** жіберіңіз.

## 2. Pull Request жіберу қадамдары

1. Жобаны **Fork** жасаңыз.
2. Жаңа тармақ (**Branch**) құрыңыз: `git checkout -b feature/amazing-feature`.
3. Өзгерістерді енгізіп, **Commit** жасаңыз: `git commit -m 'Add some amazing feature'`.
4. Тармақты **Push** жасаңыз: `git push origin feature/amazing-feature`.
5. **Pull Request** ашыңыз.

## 3. Код жазу ережелері

* **TypeScript** қолданыңыз.
* Кодты **Prettier** және **TSLint** арқылы тексеріңіз.
* Функциялар мен айнымалыларға түсінікті атаулар беріңіз.

Біз бірге қазақ тіліндегі теру жылдамдығын арттыратын ең жақсы платформаны жасаймыз! ✨
